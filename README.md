# BeFX Studios â€“ Bug Reporting Guidelines

This repository is dedicated exclusively to reporting bugs and issues related to any BeFX Studios add-on.  
Please follow all the instructions below when opening an issue.

---

## Required Information for Every Bug Report

1. **Screenshots**
   - A screenshot of the visible error in the interface.
   - A screenshot of the Blender console showing the complete error message.

2. **System Information**
   - Operating system.
   - Blender version.
   - BeFX Studios add-on version.

3. **Detailed Steps**
   - A clear, step-by-step description of the actions performed before the error occurred.

4. **Project Files**
   - Upload the `.blend` file (and any necessary textures) to a file host such as WeTransfer, Google Drive, Dropbox, or similar.
   - Include a download link in your issue with proper access permissions.

---

## Issue Structure

When creating an issue, please include:

- Short description of the problem  
- System information  
- Step-by-step reproduction steps  
- Current result  
- Expected result  
- Screenshots of the error and Blender console  
- Download link to the project file  
- Any additional relevant information  

---

Following these guidelines ensures that issues can be reproduced, investigated, and resolved efficiently. Thank you for your cooperation.
